GitHub upload instructions
==========================

This folder is ready for GitHub Pages.

What to upload:
- Upload the CONTENTS of this folder to the root of your GitHub repository.
- Keep the folder structure exactly as-is.
- The main entry point is index.html.

Included:
- index.html: landing page with links to both versions.
- version_fade_search/: fading / rotating people version.
- version_all_visible_search/: all people visible version.
- local assets and people folders copied into each version.
- populated_people_used.csv: reference table of the people used.
- image_manifest.csv: image/source reference.

Notes:
- This is the STRICT 43-person version using only clear location evidence.
- All image paths used by the page are relative local paths and were checked.
- Do not upload only the inner index.html file by itself; upload the folders too.
- For GitHub Pages, enable Pages from repository settings, branch main, root folder.
